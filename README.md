Projet de simulation – Développement Python 

📌 Contexte

Ce projet a été réalisé dans le cadre d’un travail académique en groupe (3 étudiants).
L’objectif était de concevoir et développer une simulation intelligente d’un carrefour à feux tricolores, 
capable de s’adapter à différents scénarios de circulation.

Ce projet met en avant des compétences en :

 _ Programmation orientée objet (POO)

 _ Architecture logicielle modulaire

 _ Simulation temps réel

 _ Gestion d’états et de transitions

 _ Logique algorithmique

 _ Développement d’interface graphique

🎯 Objectifs du projet

 _ Modéliser un système réaliste de gestion de carrefour

 _ Implémenter plusieurs scénarios dynamiques

 _ Assurer la sécurité (anti-collision, gestion d’intersection)

 _ Séparer clairement la logique métier, l’interface et la simulation

⚙️ Technologies utilisées

 _ Python 3

 _ Programmation Orientée Objet (POO)

 _ Bibliothèque graphique (Turtle / Tkinter selon votre implémentation)

 _ Gestion d’événements

 _ Architecture modulaire

🚦 Scénarios implémentés

1️⃣ Mode Normal

Durées équilibrées

Trafic modéré

Fonctionnement standard

2️⃣ Heure de Pointe

Vert prolongé

Orange réduit

Rouge raccourci

Forte densité de véhicules

Files d’attente visibles

3️⃣ Mode Nuit

Feu orange clignotant

Suppression du cycle rouge/vert

Faible densité

Priorité à droite

Circulation fluide

🧠 Compétences développées

Conception orientée objet

Gestion d’un système multi-composants

Synchronisation logique entre entités

Simulation temps réel

Organisation de projet collaboratif

Structuration de code maintenable
